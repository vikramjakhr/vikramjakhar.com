Go has an interesting approach to parse string object to time and time object to string. Go supports time formatting and parsing via pattern based layouts. Go's time package provides all the necessary constants, functions and methods to work with time and date in any go program.

» Current time (Now)

package main

import (
	"fmt"
	"time"
)

func main() {
	// Get the current time.
	t := time.Now()
	fmt.Println(t)

	// Print year, month and day of the current time.
	fmt.Println(t.Year())
	fmt.Println(t.Month())
	fmt.Println(t.Day())
}

Output : 

2016-09-17 13:55:40.557941309 +0530 IST
2016
September
17

» Creating time/date using Time.Date

In Go you can create custom time by using time.Date function. 

func Date(year int, month Month, day, hour, min, sec, nsec int, loc *Location) Time

Here is the example that creates custom time.

func main() {
	// Create a Date in 1992.
	t := time.Date(1992, 12, 26, 12, 0, 0, 0, time.UTC)
	fmt.Println(t) // 1992-12-26 12:00:00 +0000 UTC
}

» Points to remember while working with time

In go you can format or parse a time to one which you want. To achive this go has a standard time which is of like : 

Monday, 02-Jan-06 15:04:05 MST

You have to use this statndard time to format or parse any time. You can also use some of the predefined layouts in time package with Time.Format or Time.Parse to avoid remembering the standard time.

time.ANSIC // Mon Jan _2 15:04:05 2006
time.UnixDate // Mon Jan _2 15:04:05 MST 2006
time.RubyDate // Mon Jan 02 15:04:05 -0700 2006
time.RFC822 // 02 Jan 06 15:04 MST
time.RFC822Z // 02 Jan 06 15:04 -0700 // RFC822 with numeric zone
time.RFC850 // Monday, 02-Jan-06 15:04:05 MST
time.RFC1123 // Mon, 02 Jan 2006 15:04:05 MST
time.RFC1123Z // Mon, 02 Jan 2006 15:04:05 -0700 // RFC1123 with numeric zone
time.RFC3339 // 2006-01-02T15:04:05Z07:00
time.RFC3339Nano // 2006-01-02T15:04:05.999999999Z07:00
time.Kitchen // 3:04PM
// Handy time stamps.
time.Stamp // Jan _2 15:04:05
time.StampMilli // Jan _2 15:04:05.000
time.StampMicro // Jan _2 15:04:05.000000
time.StampNano // Jan _2 15:04:05.000000000

» Format a time

In go you can format a time to any format by using Time.Format method of time package. Format method takes a single argument of string type.

func (t Time) Format(layout string) string
// By using predefined layouts
t := time.Now()
fmt.Println(t.Format(time.RFC3339)) // 2016-09-17T14:25:28+05:30

// or by using you custom layout
form := "January 2, 2006"
fmt.Println(t.Format(form)) // September 17, 2016

» Parsing a string to time

Go has time.Parse function for parsing a string to time object.

func Parse(layout, value string) (Time, error)

Here is the exmple to illustrate time parsing.

value := "December 26, 2016"

// The layout must be January 2,2006.
layout := "January 2, 2006"

// Parse the string according to the layout.
t, _ := time.Parse(layout, value)
fmt.Println(t) // 2016-12-26 00:00:00 +0000 UTC

» Sleep function

Sleep function paused the execution of current goroutines for a duration of time. Sleep function takes duration as argument. 

func Sleep(d Duration)
func main() {
    for i := 0; i < 3; i++ {
	    // Duration receives nanoseconds.
	    d := time.Duration(1000000000)
	    // Sleep for 1 second.
	    time.Sleep(d)
	    fmt.Println("Current Time is : ", time.Now())
    }
}

» Subtracting one time from another

You can leverage the Time.Sub method to get duration between two times.

func (t Time) Sub(u Time) Duration

Here is the example to illustrate this.

func main() {
	t1 := time.Now()

	// Sleeps for 2 seconds.
	time.Sleep(time.Second * 2)

	t2 := time.Now()

	// Get duration between t1 and t2.
	d := t2.Sub(t1)
	fmt.Println("Duration : ", d)
	// Get seconds from duration.
	s := d.Seconds()
	fmt.Println("Seconds : ", s)
	// Get minutes from duration.
	m := d.Minutes()
	fmt.Println("Minutes : ", m)
}

Output :- 

Duration :  2.000125515s
Seconds :  2.000125515
Minutes :  0.03333542525

» Before, After and Equal

Go has also three methods named Before, After and Equal for performing comparision between different time stamps.

func main() {
	t1 := time.Date(2010, 12, 26, 12, 0, 0, 0, time.UTC)
	t2 := time.Date(2015, 12, 26, 12, 0, 0, 0, time.UTC)

	// first time is before the second.
	fmt.Println(t1.Before(t2))

	// second time is after the first.
	fmt.Println(t2.After(t1))

	// first time is after the second.
	fmt.Println(t1.After(t2))

	t3 := time.Date(2015, 12, 26, 12, 0, 0, 0, time.UTC)
	// first and second are equal
	fmt.Println(t1.Equal(t2))
	// second and third are equal
	fmt.Println(t2.Equal(t3))
}

Output :-

true
true
false
false
true
