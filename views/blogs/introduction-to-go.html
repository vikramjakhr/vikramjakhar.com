As in modern era of rapid development, developers have to make an uncomfortable choice between rapid development and high performance while choosing a language for their projects. Languages like C and C++ offer fast execution, whereas languages like Python and Ruby offers rapid development. But the Go team went beyond to solve this problem of developers and offers a high-performance language with features that make development fast.

Go (also known as golang) is an open source programming language created at Google by Robert Griesemer, Rob Pike, and Ken Thompson in 2009. It is a compiled, statically typed and many other strongest features like garbage collection, concurrency and scalable.

Go is known not only by what it includes, but also by what it doesn’t include. Go has a very concise syntax, and a compiler that’s very fast, even sometimes you’ll forget it’s running. You'll wait very less time for your project to build.

By using Go’s built-in concurrency features, you can scale your software to use the resources available without using any special threading libraries.

Go has a simple and effective type system that takes much of the overhead out of object-oriented development and lets you focus on code reuse.

Go also has a garbage collector, so you don’t have to worry much about memory. Let’s look quickly at some of these key features.

» Rapid development

Go compiles by using a smart compiler and a very simplified dependency resolution algorithm. While building a Go program, the go compiler only needs to look at the libraries that you include directly, rather than traversing the dependencies of all the libraries which are included in the entire dependency chain like C, C++ and Java. Even many Go applications compiles in few seconds in modern hardware.

» Goroutines

Go’s concurrency support is one of its strongest features. A a programmer one of the most difficult thing is to write an application that effectively uses the available resources of the hardware currently running it.

Modern computers have many cores, but most programming languages don’t have any effective tools that utilizes these resources effectively. It often requires lot of thread synchronization code, which is prone to errors.

But Go's goroutines are functions that can run concurrently with other goroutines, even including the entry point of your program. In Go you can execute many goroutines in a single thread. If you want to execute some code concurrently while you are doing other things, a goroutine is perfect choice for the job. Here’s a quick example: 

func greetSomeone(name string) {
    fmt.Println("Hi ", name)
}

And elsewhere in your code where you want to greet someone, just type 

go greetSomeone("Vikram")

» Go's Channels

Go’s channels are like data structures that allows safe data communications between goroutines. Channels helps to avoid shared memory access problem that we faces in many programming languages.

» Go's type system

Go has a flexible type system that enables code reuse with minimal refactoring overhead. In go it feels like object oriented development but without worrying about traditional approach like abstract classes and all that we see in other programming languages. Go uses composition design pattern to simply embed types to reuse functionality.

Go has both built-in types and user-defined types. Built in types like int and stringa, And a typical user-defined type in Go will look similar to structs in C and operates similarly. But Go's types can also declare methods that operate on that data.

» Memory management

Go has a modern garbage collector that does the hard work for you, so that you don't have to worry about memory related things that you needs to do in C and C++. Althogh Go's garbage collection adds little overhead to program execution time but simultaneously reduces the development efforts.


Apart from all, Go has also others feautes like strong tooling machanism which reduces development time.

To run a Go application, you have to just build and run the binary and it's all done. Nothing like jar, war and all kind of stuff.
