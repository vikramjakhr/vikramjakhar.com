Java SE 8 added a new class named StringJoiner for joining strings.

Here is what javadoc of StringJoiner says:

StringJoiner is used to construct a sequence of characters separated by a delimiter and optionally starting with a supplied prefix and ending with a supplied suffix.

As from javadoc we can conclude that after StringJoiner we can stop using libraries like Guava and string join mechanism of java prior to java SE 8.

» Constructors in StringJoiner

StringJoiner(CharSequence delimiter)
StringJoiner(CharSequence delimiter,
                 CharSequence prefix, CharSequence suffix)

» Methods in StringJoiner

StringJoiner add(CharSequence newElement)
int length()
StringJoiner merge(StringJoiner other)
StringJoiner setEmptyValue(CharSequence emptyValue)

» Creating StringJoiner Object and Adding values to it

Below is the example that joins different strings with comma.

StringJoiner stringJoiner = new StringJoiner(", ");
stringJoiner.add("vikarm").add("jakhar");
System.out.println(stringJoiner); // vikram, jakhar
System.out.println(stringJoiner.toString()); // vikram, jakhar

» StringJoiner with Prefix and Suffix

You can also create StringJoiner object by specifying prefix and suffix as :

StringJoiner stringJoiner = new StringJoiner(", ", "[", "]");
stringJoiner.add("vikarm").add("jakhar");
System.out.println(stringJoiner); // [vikarm, jakhar]
System.out.println(stringJoiner.toString()); // [vikarm, jakhar]

» Using StringJoiner for overriding toString

You can leverage the benefit of StringJoiner to override toString method. Below is the new way to override toString using java SE 8.

public class Person {
    String firstName;
    String lastName;
    int age;

    public Person(String firstName, String lastName, int age) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.age = age;
    }

    //getters and setters

    @Override
    public String toString() {
        return new StringJoiner(", ",
             this.getClass().getSimpleName() + "[", "]")
            .add("firstName = " + firstName)
            .add("lastName = " + lastName)
            .add("age = " + age).toString();
    }
}

// Creating Object of Person class
Person person = new Person("vikram", "jakhar", 24);

// Person[firstName = vikram, lastName = jakhar, age = 24]
System.out.println(person.toString());
