Go provides three types of data structures to manage data collections: arrays, slices, and maps.

An array is a fixed-length data type that contains the sequence of elements of a single type. An array is declared by specifying the data type and the length.

» Declaring an Array

var x [4]int

Above line of code declares an array of 5 elements of int type

» Declaring and Initializing an Array

package main

import "fmt"

func main() {
    var x [5]int
    x[0] = 10
    x[1] = 20
    x[2] = 30
    x[3] = 40
    fmt.Println(x)
}

You should see the output as :- 

[10 20 30 40]

» Array Initialization using Array Literal

You can also initialize an array in go while declaring it.

x := [4]int{10, 20, 30, 40}

OR as a multiline statement : 

x := [4]int{
    10,
    20,
    30,
    40,
}

» Array Initialization using ...

If you don't know array size in advance, then you can also go with initialization as in code below. In this type of initialization size of array will be calculated automatically based on the number of arguments to initialization.

x := [...]int{10, 20, 30, 40}

» Initialization of Specific Elements in an Array

You can also initialize an array in go while declaring it.

package main

import "fmt"

func main() {
    x := [5]int{2: 20, 4: 40}
    fmt.Println(x)
}

You should see the following output : 

[0,0,20,0,40]
