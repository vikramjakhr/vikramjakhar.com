Go provides built-in map type that implements hash table. A map is a data structure that provides you an unordered collection of key/value pairs. As because map is based on the hash table, it retrieve data very quickly.

» Declaring & Initialization

Below is the code for declaring a map variable. It always holds nil. You cannot assign any value in a nil map.

// map[KeyType]ValueType
// A map with string as key and int as value
var m map[string]int

You can also declare map as in below code. It allocates memory to map and you can put value into it. 

// declaring with type and initializing
var num map[int]string = make(map[int]string)

// OR without declaring with type as
num := make(map[int]string)

You can also initialize a map with values while declaring it as :

num := map[int]string{1:"ONE", 2:"TWO"}

» Rules for creating map key

A map key can be any value from built-in or struct types except slices, function and any other type that contains slices. As rule of thumb always use map key that can be compared using == operator.

// map using slice of int as key
// Compiler Error : invalid map key type []int
s := map[[]int]string{}

You can use slice as value type as :

s := map[int][]string{}

» Assigning values to a map

num := map[int]string{}
num[1] = "ONE"
num[2] = "TWO"

You cannot assign value to a nil map. below is the code

var num map[int]string
num[1] = "ONE" // Error : assignment to entry in nil map

» Retrieving a map value

You can retrive map value based on key, which also gives if exist status.exists status is the boolen value based on existence of key in map.

// Retrieve value for key 1
value, exists := num[1]
// if exists key
if exists {
    fmt.Println(value)
}

You can also retrieve only value by ignoring exists part as :

value := num[1]
// And check existence as
if value != "" {
    fmt.Println(value)
}

» Iterating over map 

num := map[int]string {
    1 : "ONE",
    2 : "TWO",
    3 : "THREE",
    4 : "FOUR", // Don't forget to add , after last entry, because it's Go.
}
for key, val := range num {
    fmt.Printf("Key %d Value %s \n", key, val)
}

» Removing an item from map

You can remove an item from map based on key using delete function as :

// Remove key/value pair for the key 1
// syntex : delete(map, key)
delete(num, 1)
