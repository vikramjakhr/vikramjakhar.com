A slice is a data structure that is very similar to an array, but without any specified length. Slices are abstraction built on top of an array type that provides a more convenient way of working with collections. Unlike regular arrays where array size remains fixed, slices are dynamic in nature. Which means the length of slices can be changed at a later stage as size of data increases or shrinks. Slices are very useful when the number of elements to be stored into a collection canâ€™t be predicted in advance.

While developing an application, you may come across many situations where you don't know size of data in advance, in such situations it's bettre to use slice.

» Declaring a Nil Slice

var x []int

In above code x is declared as a slice without any specified length. As because slices are dynamic in nature you can modify their length anytime.

» Creating an Empty Slice

x := []int{}
len(x) // 0
cap(x) // 0

Here x is declared as a empty slice whose length and capacity both are 0.

» Using make Function to Create a Slice

x := make([]T, length, capacity)  // T stands to element type.
x := make([]int, 10, 20)
x := make([]int, 10)

While declaring a slice using make function you need to specify length and an optional capacity. If the slice capacity is not specified(as in the last line in above code) then capacity will be the same as length.

» Creating a Slice with Slice Literal

Creating a slice using slice literal is same as in Array. While declaring a slice we need not to specify length within [ ] operator. The initial length and capacity will automatically taken from the number of elements that are initialized. 

x:= []int{10, 20, 30, 40}

» Slice with the append function

The append function creates a new slice by taking existing slice and appending all the elements to it. If the capacity of the slice is not sufficient to hold the new values, append function grows capacity of slice automatically. 

package main

import "fmt"

func main() {
    x := []int{10,20,30}
    y := append(x, 40, 50)
    fmt.Println(x, y)
}

You should see output as :- 

[10 20 30] [10 20 30 40 50]

» Slice with the copy function

As the name suggests copy function creates a new slice by copying elements of an existing slice to a new slice. 

package main

import "fmt"

func main() {
    x := []int{10, 20, 30, 40}
    y := make([]int, 3)
    copy(y, x)
    fmt.Println(x, y)
}

You should see output as :- 

[10 20 30 40] [10 20 30]

As in the above output, slice y has [10 20 30]. Because we specified length of slice y as 2, so it copy first 2 elements from slice x.
